Control flow and loops

The control flow is the order in which the code is run/executed. Code runs from top to bottom, and we can manipulate the programs execution using conditional statements or loops. the code should u run in the right order to work- like how you wouldn't go outside before putting on your clothes. Loops helps us to repeat a task/function without having the write the code over and over again. We'll set a conditional to repeat this code until a desired outcome. Say we want to be fully dressed before leaving the house- the loop will repeat itself until we have put on all of our clothing items and are ready to step outside.

What is the DOM?

The DOM (Document Object Model) is the interface we use to manipulate certain elements within our web browser. The DOM is structured in a tree-like shape- having a root base, followed by multiple different branches with even more smaller branches. We use the DOM to interact with elements such as HTML elements, changing CSS and adding interactive (events) to make our webpage useable.
